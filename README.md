## 🚀 Features

*   **🖱️ Cursor Tracking:** The background dynamically reveals itself as you move your mouse across the screen.
*   **🌊 Ripple Effects:** Clicking anywhere on the canvas triggers a smooth, outward-expanding wave animation.
*   **🖼️ Switchable Backgrounds:** Built-in functionality to swap out and transition between different 1920x1080 wallpapers.
*   **⚡ Highly Optimized:** Smooth frame rates and responsive design utilizing pure frontend web technologies.

## 💻 How to Use as Your Desktop Wallpaper

Want to use this interactive background on your actual PC desktop instead of just in the browser? You can easily do this using **Lively Wallpaper**!

1. 📥 **Download the App:** Open the **Microsoft Store** on your Windows PC, search for **Lively Wallpaper**, and install it (it's completely free).
2. ⚙️ **Launch Lively Wallpaper:** Open the application from your Start menu.
3. ➕ **Add New Wallpaper:** Click the **"+" (Add Wallpaper)** button in the app.
4. 📂 **Select Your File:** Click **Browse**, navigate to the folder where you downloaded this project, and select the `index.html` file. 

*Boom! Your interactive video background is now running live on your Windows desktop!*
